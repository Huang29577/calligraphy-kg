【项目背景】

本项目采用：

RDF
Apache Jena Fuseki
SPARQL
LangGraph
Flask

技术路线实现知识图谱问答系统。

我负责的部分包括：

LangGraph问答系统
SPARQL自动生成
Flask后端接口

目前后端功能已经全部打通。

系统工作流程如下：

用户提问
↓
LangGraph
↓
实体识别（Entity Node）
↓
意图识别（Intent Node）
↓
Conditional Edge 路由
↓
SPARQL生成节点
↓
Fuseki查询
↓
答案返回
↓
Flask API
↓
前端展示

目前已经实现：

Fuseki部署成功
TTL知识图谱导入成功
SPARQL查询成功
Python调用Fuseki成功
LangGraph工作流搭建成功
Conditional Edge条件路由成功
Flask接口成功

目前Flask接口：

POST /ask

请求格式：

{
"question": "文彭的字号是什么"
}

返回格式：

{
"answer": "寿承"
}

前端只需要调用该接口即可。
